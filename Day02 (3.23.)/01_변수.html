호이스팅 : 
        스코프 내부에서의 변수선언은 최상위에 선언된 것처럼 사용한다.
        {} - 최상단으로 끌어 올려진 것 같은 현상

        변수선언
        1. 선언: 변수등록 (변수는 스코프가 참조하는 대상)
        2. 초기화: 변수 공간의 메모리 확보 -> undefined(값이 없는 상태)
        3. 할당: 값을 할당

        var
        1. 선언 및 초기화
        2. 할당

        let
        1. 선언
        2. 초기화
        3. 할당

        const
        1. 선언 + 초기화 + 할당

        var: 함수 스코프의 영역 인식
        let, const: 함수스코프, 블럭스코프의 영역 인식
        function() {}
        if{} for{}... <- 이렇게 블럭이 붙는 것이 블럭 스코프
