반복문: 연속으로 횟수만큼 반복처리 한다

        1. for - 주로 횟수가 정해져 있을 때 사용
           for(초기값; 최종값; 증감식) // 세 문장을 하나로 합쳐놓은것이라 ; 사용
           * for( ; ; ): 무한반복 

        2. while - 주로 무한반복 할 때 (개발단에서 많이 씀)
           초기값
           while(조건식){
               증감식
               실행문
           }
           * while(true) : 무한반복
        3. do~while - 무조건 한번 비교 (쓰는 사람만 씀)

        ---------------------------------------
        for( 초기값; 최종값; 증감식){
            실행문
        }
        초기값: 시작값 주로 i, j , 순으로 작성한다 
        (처음에는 문법이 i로 정해져있었으나 지금은 아님, 그러나 암묵적인 룰로 사용)
        최종값: 조건식 ~까지 true/false
        증감식: ++,  --
