if문: 만약에 ~이면
        예) 만약에 수업 끝나고 친구를 만나면 저녁을 먹는다
        if(친구를 만나면){
            저녁을 먹는다
        }

        *결과가 1줄 이면 {} 생력가능(근데 내용많으면 헷갈림)
        if(친구를 만나면)저녁을 먹는다;

        
        if(조건식): 관계, 논리연산자 - true/false + 산술연산자        
        if(조건식){
            true결과
        } 

        예) 만약에 수업 끝나고 친구를 만나면 저녁을 먹는다 그렇지않으면 집에 간다
        if(친구를 만나면){
            저녁을 먹는다
        }else{
            집에 간다
        }

        if(조건식){
            true결과
        }else{
            false결과
        }


        다중if문
        결과 갯수 - 1 : if문 갯수
        철수를 만나면 커피를 마시고
        영희를 만나면 술을 마시고
        재석이를 만나면 고기를 먹고
        호동이를 만나면 저녁을 먹고
        아무도 안만나면 집에 갈것이다.
        (if문 갯수: 4개)

        if(철수를 만나면){
            커피를 마시고
        }else if(영희를 만나면){
            술을 마시고
        }else if(재석이를 만나면){
            고기를 먹고
        }else if(호동이를 만나면){
            저녁을 먹고
        }else{
            집에 갈 것이다.
        }
