객체
        객체.속성(키)
        객체.메소드()

        객체: 사람이 보고 만지고 느낄 수 있는 모든 대상
        속성(키): 고유값 (처음부터 가지고 있던 속성)
        메소드: 행동

        강아지: 객체
        강아지.속성 : 종, 성별, 털색
        강아지.메소드(): 뛰기, 걷기, 구르기
        강아지.함수() 라고도 한다

        함수(): 객체를 갖고있지 않다

        객체모델링: 현실 세계의 객체를 컴퓨터 언어로 변경하는 것
        키(속성): 객체가 갖고 있는 고유한 특성의 기술
        메서드: 객체가 갖고 있는 동작

        싱글 리터널 객체
        let(const)객체명 = {키:값}
        
        객체명.키

        let 객체명 = {
            키1: 값,
            키2: 값,
            키3: 값,
            메서드명: function(){
                실행문
            }
        }

        객체명.키1 / 객체명['키1']
        객체명.키2 / 객체명['키2']

        객체명.메서드()
