String.prototype.includes()
        하나의 문자열이 다른 문자열에 포함되어 있는지를 판별하고, 결과를 true 또는 false 로 반환합니다.

        구문
        str.includes(searchString[, position])

        매개변수
        searchString
        이 문자열에서 찾을 다른 문자열.
        position Optional
        searchString을 찾기 시작할 위치. 기본값 0.

        반환값
        문자열을 찾아내면 true. 실패하면 false.
