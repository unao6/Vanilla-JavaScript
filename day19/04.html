JavaScript는 흔히 프로토타입 기반 언어(prototype-based language)라 불립니다
    프로토타입(원형) 기반으로 새로운 객체 생성해가는 프로그래밍 방식 
    생성된 객체는 자신의 프로토타입 갖는다 

    프로토타입은 객체에 보이지 않는 숨겨진 곳에 값을 넣어놓고 값을 꺼내서 쓸수 있게 해준다 
    메서드들... 
    .map , push .. 등등 
    const str ='hi'
    console.dir(str) 
    console.dir(str.__proto__)

    const obj = {}
    const arr =[]
    const fun = () => {}
    console.dir( obj.__proto__) 던더프로토 

    console.dir(Array.prototype)

    .__proto__ 접근제어자 
    prototype을 접근할수 있다
